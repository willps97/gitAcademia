Comandos Básicos


1) clonagemdo repositório a primeira vez
git clone<endereço>

2) enviando códigos construídos na estação de trabalho
git add .
git commit -m "mensagem para a equipe"
git push

3) buscando códigos do servidor para a estação de trabalho.
git pull
